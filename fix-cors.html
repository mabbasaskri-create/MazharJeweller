Agar upar kaam na kare to Cloud Shell use karo

Yeh bilkul FREE hai. Koi payment nahi, koi card nahi.

  1. Kholo: https://shell.cloud.google.com
  2. Upar right mein "Mazhar Jewellers" project select karo
  3. Neeche terminal mein yeh command do:
gcloud storage buckets describe gs://mazhar-jewellers-c502a.firebasestorage.app

Pehle yeh chalao. Agar "404" aaye to yeh try karo:

gcloud storage buckets describe gs://mazhar-jewellers-c502a.appspot.com

Jo bucket exist kare, uske liye yeh chalao:

gsutil cors set cors.json gs://BUCKET-NAME-HERE

Ya ek hi command mein:

curl -X PATCH -H "Authorization: Bearer $(gcloud auth print-access-token)" -H "Content-Type: application/json" -d '{"cors":[{"origin":["https://mazhar-jeweller.vercel.app"],"responseHeader":["*"],"method":["GET","POST","PUT","DELETE","HEAD"],"maxAgeSeconds":3600}]}' https://storage.googleapis.com/storage/v1/b/BUCKET-NAME-HERE
